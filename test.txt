1. On Your Local Machine (Git Setup)
Run these in your project folder to push the code to GitHub.

Bash

git init
git add .
git commit -m "web deployment commit"
git remote add origin <your-github-repo-url>
git push -u origin main
2. On Your Local Machine (Connect to AWS)
Run this to log in to the server.

Bash

ssh -i "webapplication.pem" ubuntu@<your-ec2-public-ip>
3. Inside the AWS EC2 Terminal (Installation & Deployment)
Run these one by one to install Docker, pull code, and run the app.

Bash

# Install Docker and Nano
sudo apt-get update
sudo apt-get install docker.io
sudo apt-get install nano

# Clone the repository
git clone <your-github-repo-url>

# Navigate to the folder
cd WebDeployment

# Create the Dockerfile
nano Dockerfile
(Inside the Nano editor, type these lines):

Dockerfile

FROM nginx:alpine
COPY /usr/share/nginx/html
(Press Ctrl+O then Enter to save, and Ctrl+X to exit)

Bash

# Build the Docker Image
sudo docker build -t html-webapplication .

# Run the Container
sudo docker run -d -p 80:80 html-webapplication:latest
